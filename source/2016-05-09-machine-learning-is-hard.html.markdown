Public Service Announcement: Machine learning is harder than you think.

I spent 2 years trying to build a consumer facing machine learning product and failed.  Getting a machine to learn takes tons of data, tons of experimentation and tons of data cleaning.  Machine learning is a black hole that can and probably will consume all of your resources, expectations, hopes and dreams.

We actually got our personalized content recommendation system to do actual machine learning.  The moment it worked was kind of amazing.  We couldn't predict exactly how the support vector machines were making decisions but there it was finding needles in the haystack like we had hoped.  

The only problem was that we spent all of our time on the machine learning part and no time figuring out the product that all of this technology was supposed to power.  In the end we never figured that out and oh yea the world financial markets collapsed so that was that.

The harsh reality of my first startup: we shouldn't have gone all in on machine learning.  We should have built a product first.  We could have made quicker progress using statistics and figuring out the features.  Machine learning could have been introduced at a later point or perhaps not at all.

My advice: stick with stats for as long as you can and only use machine learning as a last resort.
