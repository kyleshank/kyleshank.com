[Formal Method](http://formalmethod.net) has released [FMNowPlayingBarButtonItem](https://github.com/formal-method/FMNowPlayingBarButtonItem).

FMNowPlayingBarButtonItem is a "Now Playing" bar button item designed to work with both iOS 6 & 7.

FMNowPlayingBarButtonItem is a category implementation that adds a `nowPlayingButton` method to UIViewController so you can easily add a Now Playing button to your NavigationBar.

I extracted this code from several radio/podcast apps that I've built that have a background player that you need access to.

Follow Formal Method on [Github](https://github.com/formal-method).
