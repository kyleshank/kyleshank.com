After 2 months of using the Swift programming language full-time I can safely say that I am a fan.  It strikes the right balance between being both a dynamic and typed language.

I haven't been this pumped about a language since Ruby.  It's been a game changer for developing apps and I think it has the potential to be a really nice server (and general) language.

[Swift was open sourced](https://swift.org) this week which was huge.  Now it has a chance to build a community of developers across  platforms and create libraries that will take it to the next level.
